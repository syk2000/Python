튜플(tuple)
    리스트와 유사한 자료형 ,( 로 시작하고, ) 로 끝남
    리스트는 생성 이후, 데이터 변경이 가능, 
    튜플은 생성 이후 변경이 "불가능"
	ex) tup = (1,2,3,4)
    값이 변하지 않는 함수들은 list와 똑같이 사용 가능

패킹(packing)
    여러 값을 쉼표로 구분해서 한번에 입력시, 튜플로 저장
    이를 패킹이라고 부름
	ex) pack = 1,2,3,4    

언패킹(unpacking)
    튜플의 좌변에 튜플 내부인자와 같은 수의 변수를 
    쉼표로 구분해서 대입 연산할 시, 각각 나눠져서 저장됨
	ex)a,b,c,d = 1,2,3,4
	a = 1, b = 2, c = 3, d = 4
    (optional) 변수명에, *을 붙이면, 갯수가 달라도 남은걸
     *이 붙은 변수에 몰아서 저장하며 언패킹
     (단, 1회 언패킹시 *은 하나만 사용 가능, 나머지를 의미하므로)
	ex)
	>>> a,*b = 1,2,3,4,
	>>> a
	1
	>>> b
	[2, 3, 4]

함수(function)
    동작을 미리 저장해 놓고, 불러다 쓰는 것
    함수를 선언할 땐 
	def 함수명(매개 변수):	#매개 변수 란 함수에 전달할 값
	    <수행할 문장>
    함수의 종류
	인자(입력값)가 있는 함수
		def 함수명(매개 변수):
		    <수행할 문장>
	인자(입력값)가 없는 함수
		def 함수명():
		    <수행할 문장>
	결괏값이 있는 함수
		def 함수명(매개 변수):
		    <수행할 문장>
		    return 결괏값
    return 
	return은 함수의 결과값 출력을 의미하며, 동시에 함수가 종료됨을 의미함
